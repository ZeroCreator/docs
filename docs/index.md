# 📚 Центральный справочник проектов

Добро пожаловать в единую точку входа для документации всех проектов.

## 📂 Доступные проекты

### [ssk-scraper](./ssk-scraper/PROJECT_DESCRIPTION.md)
Скрапер для сбора данных. Включает:
- Настройку браузера
- Работу с контейнерами
- Миграцию с Celery на Temporal
- Локальное тестирование

## 🔍 Как пользоваться

- Используйте **поиск** в правом верхнем углу
- Навигация по разделам находится **слева**
- Переключайте тему (светлая/темная) с помощью иконки вверху

## 🚀 Добавление новых проектов

1. Скопируйте папку с `.md` файлами в `docs/`
2. Добавьте раздел в `mkdocs.yml` в секцию `nav:`
3. Запустите `mkdocs serve` для проверки
